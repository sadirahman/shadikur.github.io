Teaching

As an Adjunct Faculty at York University, I teach core undergraduate courses that build the foundation of modern web development and user-centered design. My teaching focuses on helping students master full-stack technologies including HTML5, CSS, Bootstrap, JavaScript, Node.js, and MySQL as well as essential principles in user interface design and usability evaluation. I am dedicated to creating an engaging, hands-on learning environment where students develop real-world problem-solving skills, build responsive applications, and gain confidence in both technical implementation and design thinking.

Courses at York University

  • ITEC 3020: Web Technologies (Undergraduate)

    Focuses on full-stack web development using HTML5, CSS, Bootstrap, JavaScript, Node.js, and MySQL. Students design and implement responsive, data-driven websites while exploring web standards, accessibility, and secure deployment workflows through hands-on projects.

  • ITEC 3230: Designing User Interfaces (Undergraduate)

    Covers user-centered design, prototyping, and usability evaluation. Students apply HCI principles to create intuitive and accessible interfaces using Figma and iterative design methods.
