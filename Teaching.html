Teaching

Courses at York University

  • ITEC 3020: Web Technologies (Undergraduate)

    Focuses on full-stack web development using HTML5, CSS, Bootstrap, JavaScript, Node.js, and MySQL. Students learn to design and implement responsive, data-driven websites while exploring web standards, accessibility, and secure deployment workflows through hands-on projects.

  • ITEC 3230: Designing User Interfaces (Undergraduate)

    Covers user-centered design, prototyping, and usability evaluation. Students apply human-computer interaction principles to create intuitive and accessible interfaces, using Figma and iterative design methods to test and refine user experiences.
